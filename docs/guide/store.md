# 果实商店

本文档介绍如何使用果实商店兑换奖励。

![果实奖励商店](/果实奖励商店.png)

## 查看余额

在果实商店页面顶部显示当前账户的果实余额。

## 兑换奖励

### 操作步骤

1. 进入「果实商店」
2. 浏览可兑换的奖励列表
3. 点击想要的奖励卡片
4. 确认兑换数量
5. 点击「立即兑换」

### 兑换记录

![果实获取记录](/果实获取记录.jpg)

兑换成功后，可以在「我的兑换记录」中查看：
- 兑换的奖励名称
- 兑换时间
- 消耗的果实数量

## 获取果实

### 完成任务

主要的果实获取方式是完成家长设置的任务：

1. 每日打卡
2. 家长审核通过
3. 根据目标设置获得基础果实
4. 家长可额外奖励果实

### 果实记录

点击「获取记录」可以查看：
- 每次获得果实的时间
- 对应的任务名称
- 获得的果实数量

## 家长设置奖励

家长可以在「奖品管理」中：
- 添加新的奖励
- 设置奖励名称和图片
- 设置所需的果实数量
- 启用/禁用奖励
