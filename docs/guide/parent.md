# 家长功能

本文档介绍家长专属功能的使用方法。

## 进入家长中心

点击底部导航栏的「家长中心」进入家长管理页面。

![个人管理中心](/个人管理中心.jpg)

## 审核任务

### 审核流程

![家长审核](/家长审核.png)

1. 进入「家长中心」
2. 点击「待审核」标签
3. 查看孩子提交的打卡记录
4. 选择通过或拒绝：
   - **通过**：孩子获得果实奖励，树木成长
   - **拒绝**：填写拒绝原因，孩子会收到通知
5. 打卡如果比较优秀，可以额外设置奖励果实

### 额外奖励

审核时可以为孩子提供额外果实奖励：
- 使用「+」/「-」按钮调整额外果实数
- 或直接输入数量
- 系统会显示总果实数

## 目标管理

在家长中心可以：
- 为孩子创建新的习惯目标
- 编辑现有目标设置
- 删除不需要的目标

## 奖品管理

在「奖品管理」中可以：
- 添加新奖品
- 设置奖品名称和描述
- 设置所需的果实数量
- 上传奖品图片
- 启用/禁用奖品

## 切换儿童模式

家长可以开启儿童模式限制孩子的操作：
- 开启儿童模式需要密码确认
- 儿童模式下隐藏编辑功能
- 退出儿童模式也需要密码确认

## 修改密码

在个人中心可以修改登录密码：
- 需要先验证当前密码
- 输入新密码并确认
