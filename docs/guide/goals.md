# 目标管理

本文档介绍如何创建和管理习惯目标。

![设置新目标](/设置新目标.png)

## 创建目标

### 操作步骤

1. 进入「森林主页」
2. 点击右下角的「+」添加按钮
3. 填写目标信息：
   - 目标名称：如"早起"、"阅读"、"运动"等
   - 选择树木图标
   - 目标周期：每周几天
   - 每次完成获得的果实数
4. 点击「保存」

### 目标设置选项

| 选项 | 说明 |
|------|------|
| 目标名称 | 习惯的名称，如"每天阅读30分钟" |
| 树木图标 | 选择一个代表该习惯的树木 |
| 目标周期 | 每周需要完成的天数 |
| 果实奖励 | 每次完成获得的果实数量 |

## 编辑目标

在森林主页点击目标卡片，进入目标详情页，可以：
- 修改目标名称
- 调整目标周期
- 修改果实奖励数量

## 删除目标

在目标编辑页面底部点击「删除目标」，确认后即可删除。

**注意**：删除目标会同时删除该目标下所有的打卡记录，请谨慎操作。

## 多孩子管理

应用支持管理多个孩子的目标：

1. 在个人中心点击「切换孩子」
2. 选择要管理的孩子
3. 即可查看和管理该孩子的目标
