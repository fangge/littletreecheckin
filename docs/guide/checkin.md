# 如何打卡

本文档介绍如何在成就丛林应用中完成每日打卡。

![每日打卡](/每日打卡.jpg)

## 打卡流程

### 1. 进入打卡页面

点击底部导航栏的「打卡」按钮，进入打卡页面。

### 2. 选择目标

在打卡页面，您可以看到所有需要完成的目标列表。每个目标显示：
- 目标名称和图标
- 当前进度
- 已连续打卡天数

### 3. 提交打卡

点击目标卡片进入打卡详情页，您可以：
- 选择打卡日期（默认今天，也支持补打卡）
- 填写打卡备注（可选）
- 点击「打卡」按钮提交

### 4. 等待审核

打卡提交后，需要等待家长审核：
- 审核通过：树木成长，获得果实奖励
- 审核拒绝：收到通知，可以重新打卡

## 补打卡

如果您错过了某天的打卡，可以在第二天进行补打卡：

1. 进入打卡页面
2. 点击目标卡片
3. 点击日期选择器，选择历史日期
4. 填写打卡信息并提交

## 果实奖励

每次审核通过后，您将获得：
- **基础果实**：每个任务固定奖励
- **额外果实**：家长可额外奖励

果实会自动添加到您的账户，可在果实商店兑换奖励。
