# 勋章系统

本文档介绍成就丛林中的勋章系统。

![成就勋章墙](/成就勋章墙.jpg)

## 勋章类型

### 累计任务类勋章

根据完成的总任务数量解锁：

| 勋章 | 条件 |
|------|------|
| 初学者 | 完成 10 个任务 |
| 进阶者 | 完成 50 个任务 |
| 高手 | 完成 100 个任务 |
| 大师 | 完成 500 个任务 |
| 传奇 | 完成 1000 个任务 |

### 连续打卡类勋章

根据连续打卡天数解锁：

| 勋章 | 条件 |
|------|------|
| 第二天 | 连续打卡 2 天 |
| 一周坚持 | 连续打卡 7 天 |
| 半月之星 | 连续打卡 15 天 |
| 月度冠军 | 连续打卡 30 天 |
| 季度达人 | 连续打卡 90 天 |
| 年度传奇 | 连续打卡 365 天 |

### 特殊成就勋章

| 勋章 | 条件 |
|------|------|
| 早起鸟 | 早上 7 点前打卡 |
| 夜猫子 | 晚上 10 点后打卡 |
| 第一个目标 | 创建第一个目标 |

## 查看勋章

进入「勋章」页面可以查看：
- 已获得勋章（带获取日期）
- 未获得勋章（显示解锁条件）
- 勋章详情和获得时间
