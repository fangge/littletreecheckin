# 功能介绍

成就丛林是一款游戏化儿童习惯养成应用，通过游戏化的方式帮助孩子养成良好的习惯。

## 核心功能

### 🌳 森林主页

森林主页是应用的核心展示页面，以可视化的方式展示孩子培养的"习惯之树"。

![森林主页](/成就丛林.jpg)

**主要功能：**
- 树木成长可视化，每棵树代表一个习惯目标
- 支持本月 / 上季度 / 过去一年统计筛选
- 点击树木可查看详情和打卡记录
- 今日已打卡的目标会显示绿色徽章

### ✅ 每日打卡

孩子每日完成任务后进行打卡提交，由家长审核后生效。

![每日打卡](/每日打卡.jpg)

**主要功能：**
- 支持选择当天或历史日期补打卡
- 打卡后可选择图片或文字记录
- 家长审核通过后树木自动成长
- 每次完成任务获得果实奖励

### 🎯 目标设置

家长可为孩子创建和管理多个习惯目标。

![设置新目标](/设置新目标.png)

**主要功能：**
- 创建/编辑/删除习惯目标
- 设置目标名称、图标、周期等
- 支持多孩子管理
- 可自定义每次完成任务的果实奖励数量

### 🏅 勋章系统

孩子坚持完成任务可获得各种勋章激励。

![勋章墙](/成就勋章墙.jpg)

**主要功能：**
- 根据累计任务数解锁勋章
- 根据连续打卡天数解锁勋章
- 各种特殊条件成就勋章
- 勋章展示和详情查看

### 🛒 果实商店

用积累的果实兑换家长设置的实际奖励。

![果实商店](/果实奖励商店.png)

**主要功能：**
- 查看可用奖励列表
- 果实兑换记录
- 获取果实历史明细
- 家长可自定义奖励内容

### 💬 消息中心

家长与孩子互动的桥梁。

![消息中心](/鼓励消息.jpg)

**主要功能：**
- 系统自动发送审核通知
- 家长与孩子互动消息
- 任务审核状态提醒

## 特色功能

### 👶 儿童模式

家长可一键切换到儿童模式，限制孩子的操作范围。

**功能特点：**
- 隐藏编辑功能和家长管理入口
- 防止误触
- 切换需密码确认

### 🌙 深色模式

支持浅色/深色主题切换，保护眼睛。

**功能特点：**
- 适配系统偏好设置
- 手动切换主题
- 改善夜间使用体验

### 📱 PWA 支持

可安装到手机主屏幕，以全屏 App 模式运行。

**支持平台：**
- Android Chrome
- iOS Safari
- 离线缓存支持
